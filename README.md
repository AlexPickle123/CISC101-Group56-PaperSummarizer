# Research Paper Summarizer

This project implements a modular AI system prompt for summarizing academic papers.

## Project Files

### system_prompt.md
Contains the main system prompt with rules, constraints, output format, and user input requirements.

### modules/
Each file represents a functional internal system module:

- 01_intake_setup.md
- 02_section_loop.md
- 03_guardrails.md
- 04_rendering_refinement.md
- 05_citation_extractor.md
- 06_key_contributions.md

## Features
- Modular architecture
- Strict hallucination mitigation
- Output safety checks
- Expert & lay summaries
- Glossary creation
- Evidence-based summarization
